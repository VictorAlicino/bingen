BINGEN is a simple tool to generate binary files filled with 0xFF, as these is the default value for erased flash memories.

## Usage

```
$ bingen <output> <size>
```
    arguments:
        output: output file name
        size: size in bytes of the output file


## Building

```
$ make
$ sudo make install
```
